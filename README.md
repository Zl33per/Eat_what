# Eat What

一个用于解决“今天吃什么”的小工具。

## 功能

- 维护餐厅列表，每家餐厅包含地域标签和种类标签。
- 按地域和种类筛选候选餐厅。
- 从候选列表中随机挑选一家作为最终结果。
- 支持添加、删除餐厅，并将数据保存在浏览器本地。
- 默认餐厅数据单独放在 `data/restaurants.js`，可以直接编辑。

## 使用方式

直接用浏览器打开 `index.html` 即可。

## 更新餐厅数据

打开 `data/restaurants.js`，按下面格式新增或修改：

```js
window.restaurantData = [
  { name: "餐厅名称", region: "地域标签", category: "种类标签" },
];
```

页面会优先读取浏览器本地保存的数据；如果你改了 `data/restaurants.js` 后没有变化，点击页面右上角“重置数据”，就会重新使用这个文件里的餐厅列表。
